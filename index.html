Sobre o Layout

Este é um layout fictício criado por mim para estudos de desenvolvimento web, tendo sido inspirado no site da W3schools. Este é um layout responsivo, ou seja, atende aos padrões da W3C para sites exibidos em computadores desktop e dispositivos móveis. O dódigo é simples, envolve apenas o básico de html e css, utilizando o recurso float para estruturar os elmentos.
